# Margn Brainstorming Archive

Folder ini menyimpan evolusi ide Margn sebelum scope MVP diputuskan.

1. [Ide 1 — Autonomous CFO](MARGN-v1-cfo.md): CFO luas yang mengatur pricing,
   counterparty risk, negosiasi, dan strategi portfolio ASP.
2. [Ide 2 — Margin Engine](MARGN-v2-margin-engine.md): fokus pada revenue,
   inference cost, reconciliation, dan price floor.
3. [Ide 3 — Marketplace Router](MARGN-v3-router.md): membantu agent pembeli
   memilih ASP berdasarkan harga, liveness, reputasi, dan track record.
4. [Ide 4 — Profitability Copilot](MARGN-v4-profitability-copilot.md): versi
   lebih sempit dari Margin Engine yang fokus pada cost per call, profit/loss,
   price floor, simulasi harga, dan marketplace benchmark.

Keempat dokumen masih berstatus brainstorming dan belum ditetapkan sebagai arah
produk final.
